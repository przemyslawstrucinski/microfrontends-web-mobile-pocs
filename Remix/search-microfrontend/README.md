# Search Microfrontend

Microfrontend for doctor search functionality.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will run on http://localhost:3001

## Exposed Components

This microfrontend exposes:
- `SearchBar` - A search component that can be embedded in other applications

## Features

- Search doctors by name, specialty, and location
- Full search page with results
- SSR-enabled search with Remix loaders
- Mock doctor data
