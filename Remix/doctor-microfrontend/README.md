# Doctor Microfrontend

Microfrontend for displaying doctor profiles with SSR.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will run on http://localhost:3002

## Exposed Components

This microfrontend exposes:
- `DoctorProfile` - A doctor profile component that can be embedded in other applications

## Features

- Doctor profile pages with SSR
- Dynamic routes `/doctor/:id`
- Detailed doctor information
- Patient reviews
- Availability schedule
- Mock doctor data with full profiles
