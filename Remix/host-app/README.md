# Host Application

Main application that integrates the search and doctor microfrontends.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will run on http://localhost:3000

## Prerequisites

Make sure the microfrontends are running:
- Search microfrontend on port 3001
- Doctor microfrontend on port 3002

## Features

- Homepage with integrated search
- Navigation to search and doctor pages
- Module Federation integration
